Create Shader from GLSL file.

Load GLSL file and compile it into SPIR-V code in runtime using shaderc. You can specify the several compile options (e.g., target environment, optimization level, etc.) by passing the appropriate compileOptions struct, and also identifier for shader identification.

If you're looking for compile the already existing GLSL string, use vku::Shader::fromGLSLString instead.

Parameters
compilerThe shaderc compiler.
glslPathThe path to the GLSL file.
stageThe stage of the shader. Must be a kind of either vertex, tessellation control, tessellation evaluation, geometry, fragment, or compute.
compileOptionsThe compile options.
pSpecializationInfoThe specialization info of the shader. If nullptr, no specialization is used.
Returns
A RefHolder of the Shader.
Exceptions
std::runtime_errorIf the compilation failed.
Warning
If pSpecializationInfo passed, its lifetime must be tied to its actual usage (passed to vk::create{Graphics,Compute}Pipeline).
Note
This function is available only if the macro VKU_USE_SHADERC is defined.

Create Shader from GLSL string.

Compile GLSL string into SPIR-V code in runtime using shaderc. You can specify the several compile options (e.g., target environment, optimization level, etc.) by passing the appropriate compileOptions struct, and also identifier for shader identification.

If you're looking for directly load the GLSL file and process it, use vku::Shader::fromGLSLFile instead.

compileOptionsThe compile options.
pSpecializationInfoThe specialization info of the shader. If nullptr, no specialization is used.
identifierThe identifier of the shader. If not specified, it will use the function caller's source location.
Returns
