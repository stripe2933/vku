◆ inferAspectFlags()

static constexpr vk::ImageAspectFlags vku::Image::inferAspectFlags ( vk::Format format)
inlinestaticnodiscardconstexprnoexcept

Inferred image aspect flags from format.

Inferring is done by the following rules:

  • D16Unorm, D32Sfloat -> Depth
  • D16UnormS8Uint, D24UnormS8Uint, D32SfloatS8Uint -> Depth | Stencil
  • S8Uint -> Stencil
  • otherwise -> Color.
Parameters
formatFormat used for image aspect flags inferring. Must not be vk::Format::eUndefined.
