class vku::Gpu< QueueFamilies, Queues >

Bootstrapper for Vulkan RAII physical device, device, allocator creation.

Template Parameters
QueueFamiliesA struct type that would contain the queue family indices of the selected physical device.
QueuesA struct type that would contain the queue handles of the created device. It must implement the static function named getCreateInfos that is returning RefHolder of either a single vk::DeviceQueueCreateInfo or contiguous range of vk::DeviceQueueCreateInfos.
