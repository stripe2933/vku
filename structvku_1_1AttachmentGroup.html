auto vku::AttachmentGroup::getColorAttachment ( std::size_t index) const -> const Attachment&
inlinenodiscardexportnoexcept

Get the const reference of color attachment at the specified index.

Parameters
indexThe index of the color attachment.
Returns
The const reference of the color attachment.
Note
This function is designed to make it easier for users to access elements from attachmentGroup.colorAttachments[i] without needing to use the more cumbersome get<vku::Attachment>(...) call.
This function is considered "unsafe" because if the element at the specified index isn't a vku::Attachment, it can lead to undefined behavior.
