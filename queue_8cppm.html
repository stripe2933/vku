Retrieve the index of both compute and present capable queue family from queueFamilyProperties.

Parameters
physicalDeviceVulkan physical device to get queue families from.
surfaceSurface to test for presentation support.
queueFamilyPropertiesQueue family properties, which is identical to physicalDevice.getQueueFamilyProperties().
Returns
The index of the both compute and present capable queue family, or std::nullopt if not found.
Note
This is useful for compute based window rendering, because it doesn't require the explicit queue family ownership transfer between compute and present queues.
